## Usage

This generator is based on [yeoman](http://yeoman.io/). You can use it as demonstrated:

```bash
npm i -g yeoman
npm i -g generator-open-wc
yo open-wc:{generator-name}
```

or

```bash
npx -p yo -p generator-open-wc -c 'yo open-wc:{generator-name}'
```
